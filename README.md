Kareem Website

## About the Project
Kareem is a web-based platform designed to help Muslims stay consistent in their worship during Ramadan or daily life. It brings together essential tools like dzikr, dua, prayer times, and zakat calculation in one simple and accessible website.

## Why I Made This Project
I created Kareem to make it easier for Muslims to stay organized and focused on their worship, especially during Ramadan. This project aims to provide a simple solution in one place.

## How I Made This Project
This project was built using:
- HTML for structuring the website  
- CSS for styling and layout  
- JavaScript for adding interactivity and functionality

and for the illustration, i use figma

## Features

- Dzikr routine  
- Muslim dua collection  
- Prayer time  
- Zakat calculation  
- Tracker
